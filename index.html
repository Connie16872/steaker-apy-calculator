注意事項：

  1. 年化利率 APY 欄位不需輸入 %。
  2. 年化利率 APY 適用時機：方案利息以「複利」計算。
  3. 約期天數依照申購當日起算。
    舉例：2/1 開始投放，約期 3 個月，實際天數為 28+31+30=89 天。
  4. 期滿後可獲得金額的公式：
    初始資金 * ( 1 + APY% ) ^ ( 約期天數 / 365 )
    舉例：方案 A：初始本金 1000 USDT，投放天數 30 天，APY=15%；
    期滿可獲得：1000*(1+15%)^(30/365)= 1011.553515 USDT (本金加利息)
